## 프로젝트 목표

클라우드 인프라 프로비저닝을 자동화하여 배포 시간을 50% 단축합니다.

## 기술 스택

| 구성요소 | 기술 |
|----------|------|
| IaC | Terraform v1.8 |
| CI/CD | GitHub Actions |
| 컨테이너 | Docker + Kubernetes |
| 모니터링 | Prometheus + Grafana |

## 주요 마일스톤

- **Q1**: Terraform 모듈 설계 완료
- **Q2**: CI/CD 파이프라인 구축
- **Q3**: 프로덕션 마이그레이션
- **Q4**: 모니터링 대시보드 구축
