# 가이드

이 섹션에서는 Tech Docs Portal에서 문서를 작성하고 관리하는 방법을 안내합니다.

## 포함 내용

| 문서 | 설명 |
|------|------|
| MDX 작성 가이드 | MDX 문법, 코드블록, 테이블 등 작성 요령 |
| 문서 관리 가이드 | 디렉토리 구조, 페이지 추가, 버전 관리 |

## 시작하기

- 새 문서를 작성하려면 [MDX 작성 가이드](/docs/2025/guide/mdx-writing)를 참고하세요.
- 프로젝트/폴더를 추가하려면 [문서 관리 가이드](/docs/2025/guide/docs-management)를 참고하세요.
