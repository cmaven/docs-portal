---


이 섹션에서는 Tech Docs Portal에서 문서를 작성하고 관리하는 방법을 안내합니다.

## 포함 내용

| 문서 | 설명 |
|------|------|
| [MDX 작성 가이드](/docs/guide/mdx-writing) | MDX 문법, 코드블록, 테이블, Mermaid 다이어그램 |
| [컴포넌트 가이드](/docs/guide/components) | Callout, Tabs, Steps, Badge 등 UI 컴포넌트 |
| [커스터마이즈 가이드](/docs/guide/customization) | 테마, 색상, 폰트, 레이아웃 커스터마이징 |
| [문서 관리 가이드](/docs/guide/docs-management) | 디렉토리 구조, 페이지 추가, 버전 관리 |

## 시작하기

- 새 문서를 작성하려면 [MDX 작성 가이드](/docs/guide/mdx-writing)를 참고하세요.
- UI 컴포넌트를 사용하려면 [컴포넌트 가이드](/docs/guide/components)를 참고하세요.
- 테마와 색상을 변경하려면 [커스터마이즈 가이드](/docs/guide/customization)를 참고하세요.
- 프로젝트/폴더를 추가하려면 [문서 관리 가이드](/docs/guide/docs-management)를 참고하세요.
